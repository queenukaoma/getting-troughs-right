The Problem

Unreliable trough data leads to unsafe dosing decisions.

Heart transplant recipients depend on precise immunosuppressant levels. When troughs aren't drawn in the correct window or last-dose times aren't documented, the lab value is misleading — leading to unnecessary dose changes, repeat draws, and delayed decisions.

Project Aim

Standardize when and how troughs are collected.

Implement an NP-led standardized trough collection protocol with last-dose documentation to improve the reliability and timeliness of immunosuppressant levels used for dosing decisions in adult heart transplant inpatients.

Who Qualifies?

Inclusion & Exclusion Criteria

Every shift, ask: does this patient qualify for the trough timing protocol?

Include
✅ Adult (age ≥ 18)
✅ Heart transplant recipient (OHT)
✅ Admitted to ICU, IMC, or Telemetry
✅ On tacrolimus, sirolimus, cyclosporine, or everolimus
✅ Trough level ordered today
Exclude
❌ OR day or code event (timing not feasible)
❌ Clinically unstable / NPO / pre-procedural
❌ Transferred off participating unit mid-draw
❌ Both actual and planned dose times missing

The Intervention

Standardized Trough Collection Protocol

A simple, fixed workflow that removes variability from trough collection.

By 07:30
Check: is today's trough order present? If not, page the transplant provider.
08:15 – 08:45
Draw the trough level. Target time: 08:30. This is the C0 window — 0 to 60 minutes before the next dose.
By 09:00
Administer the scheduled immunosuppressant dose immediately after the draw. Do not hold the dose for the result.
MAR Documentation
Record the actual clock time the previous dose was given (the "Given" time, not the scheduled time). This is the last-dose documentation.
Escalation
If the window is missed (it's past 09:00 with no draw), if night shift drew with AM labs, or if there's any dosing ambiguity — contact the transplant provider immediately.
🚫
Do NOT draw troughs with routine 03:00–04:00 AM labs
⏲️
Do NOT hold the dose waiting for the lab result

Key Findings

What the data tells us — and why the protocol will work.

Night Shift is the Primary Driver

14 draws occurred between 03:00–04:00 AM with routine morning labs. These are never true troughs — they're 5+ hours early. The protocol's fixed 08:30 window eliminates this entirely.

Nearly 3 Hours to a Dose Decision

Median time from result posting to provider order is 167 minutes. When troughs are drawn at 08:30 and results post by 10:00, providers can act before rounds end — cutting this window significantly.

Documentation is Already Strong

97.1% of draws already have last-dose documentation. The protocol reinforces an existing strength and standardizes it — this isn't a behavior change, it's a workflow lock-in.

The Gap is Closeable

On-time trough rate is 65.7% — only 4.3 percentage points from the 70% target. 37 of 75 non-excluded draws already land in the 08:15–08:45 window. With a standardized protocol, this gap is highly closeable.

Repeat Draws = Wasted Resources

5 repeat draws were ordered because of timing issues (4.8%). Each repeat draw costs nursing time, lab resources, and delays clinical decisions. The protocol targets this directly.

Zero Dose-Change Reversals

No dose changes were reversed within 72 hours during baseline. This is a positive finding that establishes a clean starting point — the protocol aims to keep it that way as trough accuracy improves.
